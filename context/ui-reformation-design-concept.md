What to improve visually
1. Background / shell concept
Current issue

The outer beige/cream area is okay, but it does not yet sell the “notebook opened” concept.

Better direction

Use a 2-layer visual system:

Layer A — App background

A soft desk-like or warm neutral background.

Example feel:

warm beige
desaturated cream
very subtle texture
calm, non-distracting
Layer B — Main dashboard page

A large white/off-white paper sheet with:

subtle horizontal lines
soft paper texture
very light shadow
rounded but not overly rounded
thin ink border

This makes the dashboard feel like an actual notebook page placed on top.

Extra improvement

Add a subtle spine or seam between sidebar and main content, so it feels like an opened notebook.

2. Sidebar should feel like the notebook’s left side
Current issue

The sidebar looks like a separate app panel.

Better direction

Make the sidebar feel like:

the notebook margin
planner tabs
binder-side controls
a personal left-side workspace
Suggestions
slightly darker paper tone than main page
softer border
less saturated button colors
navigation buttons like “paper tabs” or “planner labels”
profile area feels like a little sticker/passport badge
logout button should still stand out, but not scream
Personality idea

The sidebar can feel like the “identity and settings” side of the notebook.

3. Main page needs real paper energy
Current issue

The big dashboard area is plain and flat.

Better direction

Your main dashboard should feel like:

a clean sheet of ruled notebook paper

Since you already generated the lined paper texture, this is very useful.

But important:

Do not make the lines too strong.
The lines should be:

very subtle
low opacity
mostly visible in empty spaces
not fighting with content

This should feel like:

a clean planner sheet

not

a noisy wallpaper
4. Borders are too heavy
Current issue

The thick navy outlines make everything feel stiff and a bit old-school wireframe-ish.

Better direction

Use:

thinner borders
softer ink-like blue/charcoal
more subtle section separation
slightly softer shadows
Rule

Keep strong borders only where you really need structure.

For the notebook feeling, sections should feel drawn on a page, not boxed in like admin widgets.

5. Typography can carry a lot of the personality

You said you want:

handwritten feel
clean
personal
Ikigai / Zen motivation

That’s perfect.

Recommended font system

Use 2-font hierarchy:

A. Handwritten / notebook font

Use for:

section titles
greeting
small labels
quote-related text
B. Clean readable font

Use for:

stats values
sidebar controls
inputs
smaller body text
chart labels
Why?

If everything is handwritten, readability will drop.
The notebook feeling should come from selective handwriting, not full handwritten overload.

What “Ikigai feel” means in UI

It means the app should feel:

calm, not rushed
intentional, not cluttered
personal, not corporate
reflective, not just functional
encouraging, not pressuring
Practical ways to show that
1. Welcome area

Include subtle motivation:

welcome line
today’s intention
small quote support
2. Daily Planner emphasis

This becomes the “live with intention” feature.

3. Stats as self-awareness, not performance pressure

Avoid making the dashboard feel like an audit board.

4. Color language

Use calm accents, not loud neon UI.

5. Empty states

Even unfinished sections should feel supportive:

“Start shaping your day”
“Capture a thought”
“No memo yet — jot down what matters”

That gives warmth.
